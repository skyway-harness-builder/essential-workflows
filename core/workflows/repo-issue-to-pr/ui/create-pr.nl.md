---
short_description: Opent de pull request.
long_description: Pusht de definitieve branch en opent een concept-pull request die verwijst naar het oorspronkelijke issue, met invulling van de PR-sjabloon van de repo indien aanwezig.
---
Rondt de goedgekeurde implementatie af door een concept-pull request te openen. Deze verwijst naar het oorspronkelijke issue en vult de pull-request-sjabloon van het project in als die er is, zodat de PR klaar is voor een laatste menselijke beoordeling.
