---
short_description: Implementeert een gelabeld GitHub-issue en opent een pull request ter beoordeling.
long_description: "Start zodra een issue het label \"ready-for-sky\" krijgt: schrijft de implementatie, vraagt jouw goedkeuring, en opent daarna een pull request. Wijs af met opmerkingen om het te laten herzien."
---
Deze workflow zet een gelabeld GitHub-issue om in werkende code. Zodra een issue klaar wordt gemeld, leest hij de beschrijving, implementeert de wijziging en draait de tests van het project. Voordat er iets ter beoordeling wordt geopend, krijg je de kans om de implementatie goed te keuren of terug te sturen met feedback — de workflow herziet dan het werk en probeert het opnieuw. Eenmaal goedgekeurd, opent hij een concept-pull request die verwijst naar het oorspronkelijke issue.
