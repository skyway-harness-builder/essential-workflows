short_description: Opent de pull request.
long_description: Pusht de definitieve branch en opent een concept-pull request met een samenvatting van het plan, de implementatie en de opmerkingen uit beide goedkeuringsstappen.
---
Rondt alles af door een concept-pull request te openen. De beschrijving vat het oorspronkelijke plan samen, wat er is gebouwd, en eventuele opmerkingen uit beide beoordelingsrondes, zodat de laatste beoordelaar het volledige beeld heeft.
