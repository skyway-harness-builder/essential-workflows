long_description: Pauzeert de run en toont de sets PASS (wordt geïmporteerd), FAIL (afgewezen) en SKIP (bestaat al). Niets wijzigt de doelmap totdat je goedkeurt. Verloopt na 1 uur zonder antwoord.
